# IPFS Uploader

The file upload component for ipfs which based on [chakra ui](https://chakra-ui.com/).
Now we only support [pinata](https://www.pinata.cloud/).

# Getting started

## Installation

```npm
npm install @osaguild/ipfs-uploader
```

```yarn
yarn add @osaguild/ipfs-uploader
```

# Release to npm

```yarn
yarn rollup
yarn login
yarn publish
```
